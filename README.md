# Sorty
This application was developed for educational purposes to help students understand two programming concepts:

- Concurrency and multithreading
- Sorting algorithms

## Implemented Sorting Algorithms
* Bead
* Bogo (disabled)
* Bubble
* Cocktail
* Cycle
* Gnome
* Heap
* Insertion
* Merge
* Odd Even
* Quick
* Radix
* Selection
* Shell
